# agri-data-viz-pro-
Crop Insight is a dark-mode dashboard bridging the gap between agri-data and farming insights. Built with Streamlit and Plotly, it analyzes 2,200+ soil samples to map $N-P-K$ profiles and climate resilience. It helps agronomists visualize nutrient trends and environmental clusters for optimal crop yields. 📊🌱 #Python #AgriTech #DataViz
